- **格式**: 保持原名
- **示例**: `2026-07-02-user-authentication.md`
- **要求**: 不删除，保留完整历史

## 计划归档规则

### 1. 归档时机
- 计划所有目标达成
- 所有验证通过
- 文档已更新
- 技术债已记录

### 2. 归档内容
- 计划文档
- 进度日志
- 验证证据
- 技术债记录

### 3. 归档流程
1. **检查完成度**: 确认所有目标达成
2. **收集证据**: 收集验证证据
3. **更新文档**: 更新相关文档
4. **移动文件**: 从 `active/` 移动到 `completed/`
5. **更新索引**: 更新此索引文件
