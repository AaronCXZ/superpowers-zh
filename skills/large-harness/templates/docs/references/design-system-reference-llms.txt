# 设计系统参考

## 概述
本文档提供设计系统的参考信息，包括组件命名规则、间距字体token、状态变体、可访问性要求。

## 组件命名规则

### 命名约定
- **组件文件**: PascalCase，如 `UserProfile.tsx`
- **样式文件**: camelCase，如 `styles.module.css`
- **类型文件**: camelCase，如 `types.ts`
- **测试文件**: 与组件文件同名，如 `UserProfile.test.tsx`

### 组件结构
```
src/components/
├── ComponentName/
│   ├── index.tsx          # 组件主文件
│   ├── ComponentName.tsx  # 组件实现
│   ├── styles.module.css  # 样式文件
│   ├── types.ts           # 类型定义
│   └── __tests__/         # 测试文件
```

### 命名示例
- **按钮**: `Button.tsx`
- **输入框**: `Input.tsx`
- **卡片**: `Card.tsx`
- **模态框**: `Modal.tsx`
- **导航栏**: `Navbar.tsx`

## 可访问性要求

### 颜色对比度
- **正常文本**: 对比度至少 4.5:1
- **大文本**: 对比度至少 3:1
- **图形元素**: 对比度至少 3:1

### 键盘导航
- **Tab 顺序**: 逻辑合理的 Tab 顺序
- **焦点指示**: 清晰的焦点指示器
- **快捷键**: 提供常用操作的快捷键

### 屏幕阅读器
- **ARIA 标签**: 为所有交互元素提供 ARIA 标签
- **语义化 HTML**: 使用语义化 HTML 标签
- **替代文本**: 为图片提供替代文本

### 动画和过渡
- **减少动画**: 支持减少动画选项
- **闪烁**: 避免闪烁内容
- **自动播放**: 避免自动播放媒体

## 响应式设计

### 断点
- **移动端**: < 576px
- **平板端**: 576px - 768px
- **桌面端**: 768px - 992px
- **大桌面端**: > 992px

### 布局原则
- **移动优先**: 从移动端开始设计
- **流式布局**: 使用流式布局
- **弹性图片**: 图片自适应容器
