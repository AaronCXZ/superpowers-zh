# 参考材料目录

## 目的
提供面向模型的参考材料，帮助 agent 理解和操作项目。

## 参考材料列表

### 1. 设计系统参考
- **文件**: `design-system-reference-llms.txt`
- **内容**: 组件命名规则、间距字体token、状态变体、可访问性要求
- **用途**: 指导前端组件开发

### 2. 部署参考
- **文件**: `nixpacks-llms.txt`
- **内容**: 部署/打包规则摘要：build入口、runtime假设、环境变量要求、常见失败特征
- **用途**: 指导部署配置

### 3. 包管理参考
- **文件**: `uv-llms.txt`
- **内容**: Python包管理参考：install/sync命令、lockfile策略、virtualenv约定、verification命令
- **用途**: 指导Python项目依赖管理

## 参考材料格式

### 文本格式
- 使用纯文本格式
- 包含清晰的标题和分类
- 提供具体的示例

### 内容结构
```
# [参考材料标题]

## 概述
简要描述参考材料的内容和用途。

## 详细内容
[具体内容]

## 示例
[具体示例]

## 注意事项
[使用注意事项]
```

## 参考材料管理

### 1. 创建参考材料
- 从现有文档提取关键信息
- 整理成适合 agent 阅读的格式
- 添加必要的示例和说明

### 2. 更新参考材料
- 定期检查参考材料的准确性
- 根据项目变化更新内容
- 添加新的参考材料

### 3. 维护参考材料
- 删除过时的参考材料
- 修复错误的参考材料
- 优化参考材料的可读性

## 使用指南

### 对于 agent
1. 阅读相关参考材料
2. 理解参考材料的内容
3. 应用参考材料到实际工作中

### 对于开发者
1. 创建和维护参考材料
2. 确保参考材料的准确性
3. 分享有用的参考材料

## 扩展参考材料

### 添加新参考材料
1. 确定参考材料的主题
2. 收集相关内容
3. 整理成参考材料格式
4. 添加到此目录
5. 更新此索引

- 项目文档
